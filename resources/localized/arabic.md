# موارد مجتمع BOSC

توفر هذه الصفحة دليلاً سريعاً للمساهمين والمتعلمين الذين يفضلون اللغة العربية. هدفنا هو جعل معرفة البرمجيات مفتوحة المصدر متاحة في البيئات التعليمية والقطاع العام.

## المواضيع الرئيسية

- تراخيص المصدر المفتوح: فهم الفروق بين Apache 2.0 و MIT و GPL و Creative Commons.
- حوكمة المجتمع: استخدام GitHub issues و pull requests لاتخاذ قرارات شفافة.
- الأمان: مراجعة الروابط والتبعيات والثغرات الأمنية في المشاريع مفتوحة المصدر.

## الروابط الموصى بها

- [دليل المصدر المفتوح](https://opensource.guide/)
- [اختر ترخيصاً](https://choosealicense.com/)
- [وثائق GitHub](https://docs.github.com/ar)

## المشاركة

افتح "issue" باستخدام القالب الرسمي إذا كان المورد قديماً، أو إذا كان الرابط لا يعمل، أو إذا كنت ترغب في إضافة محتوى جديد باللغة العربية.
