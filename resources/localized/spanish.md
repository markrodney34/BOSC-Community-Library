# Recursos de la Comunidad BOSC

Esta página proporciona una guía rápida para colaboradores y estudiantes que prefieren el español. Nuestro objetivo es hacer que el conocimiento del software de código abierto sea accesible en entornos educativos y el sector público.

## Temas Clave

- Licencias de código abierto: comprenda las diferencias entre Apache 2.0, MIT, GPL y Creative Commons.
- Gobernanza comunitaria: utilice issues de GitHub y pull requests para la toma de decisiones transparente.
- Seguridad: audite enlaces, dependencias y vulnerabilidades en proyectos de código abierto.

## Enlaces Recomendados

- [Guía de Open Source](https://opensource.guide/es/)
- [Elige una Licencia](https://choosealicense.com/es/)
- [Documentación de GitHub](https://docs.github.com/es)

## Participación

Abra un issue utilizando la plantilla oficial si un recurso está desactualizado, un enlace no funciona o si desea agregar nuevo contenido en español.
