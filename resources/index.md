# Resource Index

## Categories

### 📚 Open Source Software — Foundational Reading
- [The Cathedral and the Bazaar](https://www.catb.org/~esr/writings/cathedral-bazaar/cathedral-bazaar/) — Eric S. Raymond's landmark essay on OSS development models ✅
- [Producing Open Source Software](https://producingoss.com/) — Karl Fogel's comprehensive guide to running a successful OSS project ✅
- [Open Source Guide (GitHub)](https://opensource.guide/) — Official GitHub guide for open source maintainers and contributors ✅
- [The Open Source Way 2.0](https://www.theopensourceway.org/) — Community-authored guide to open source governance and community building ✅

### ⚖️ Licensing & Legal Resources
- [Choose A License](https://choosealicense.com/) — GitHub's interactive license selection tool ✅
- [SPDX License List](https://spdx.org/licenses/) — Software Package Data Exchange — complete list of approved OSS licenses ✅
- [OSI Approved Licenses](https://opensource.org/licenses/) — Open Source Initiative's official list of approved licenses ✅
- [Apache License 2.0 (Full Text)](https://www.apache.org/licenses/LICENSE-2.0) — The license governing this project ✅
- [Creative Commons License Chooser](https://creativecommons.org/choose/) — For documentation and content licensing ✅

### 🛠️ Developer Tools & Platforms
- [GitHub Docs](https://docs.github.com/) — Full GitHub platform documentation ✅
- [Git Official Documentation](https://git-scm.com/doc) — Official Git version control documentation ✅
- [GitHub Actions Docs](https://docs.github.com/en/actions) — CI/CD automation with GitHub Actions ✅
- [Shields.io](https://shields.io/) — Dynamic badge generation for README files ✅
- [Conventional Commits Specification](https://www.conventionalcommits.org/) — Standard for structured commit messages ✅

### 🌍 Community & Governance
- [Contributor Covenant](https://www.contributor-covenant.org/) — The industry-standard Code of Conduct ✅
- [CHAOSS Metrics](https://chaoss.community/) — Community Health Analytics for Open Source Software ✅
- [Open Source Collective](https://opencollective.com/opensource) — Funding and fiscal hosting for OSS projects ✅
- [TODO Group](https://todogroup.org/) — Resources for building open source programs in organizations ✅

### 🎓 Learning Platforms & Courses
- [Linux Foundation Training](https://training.linuxfoundation.org/resources/?_sft_content_type=free-course) — Free open source courses from the Linux Foundation ✅
- [edX Open Source Development](https://www.edx.org/learn/open-source-software) — University-level OSS courses on edX ✅
- [freeCodeCamp](https://www.freecodecamp.org/) — Free coding education with a strong open source community ✅
- [The Odin Project](https://www.theodinproject.com/) — Open source web development curriculum ✅

### 🔐 Security & Compliance
- [OpenSSF Best Practices Badge](https://bestpractices.coreinfrastructure.org/) — Core Infrastructure Initiative security badge program ✅
- [OWASP Open Source Security Guide](https://owasp.org/www-project-open-source-software-top-10/) — Security risks specific to open source components ✅
- [Dependabot Documentation](https://docs.github.com/en/code-security/dependabot) — Automated dependency security updates ✅
