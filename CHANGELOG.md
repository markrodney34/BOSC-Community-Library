# Changelog

All notable changes to the BOSC Community Library are documented in this file.

## v1.3.0 - 2026-05-11

- Added GitHub Actions workflow for automated link validation in Markdown files.
- Expanded localized resource support to 5 languages (Kiswahili, French, Portuguese, Spanish, and Arabic).
- Reorganized repository structure by moving strategy and analysis documents to the `docs/` directory.

## v1.2.0 - 2026-05-11

- Added localized resource pages for Kiswahili, French, and Portuguese audiences.
- Added `resources/database.json` to support machine-readable indexing.
- Added project maintenance ownership with `CODEOWNERS`.
- Added sustainability and public-sector adoption strategy documents.

## v1.1.0 - 2026-05-11

- Audited `resources/index.md` and replaced broken or deprecated links.
- Improved project metadata, status badges, version table, and maintainer details.

## v1.0.0 - 2026-05-06

- Created the initial BOSC Community Library repository structure.
- Added issue templates, pull request template, Code of Conduct, and Contributing guide.
- Added Apache License 2.0 and legal analysis documentation.
