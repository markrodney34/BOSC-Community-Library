# Resource Index

This is the master index of all resources available in the BOSC Community Library.

## Categories

### 📚 Open Source Software
- [The Cathedral and the Bazaar](http://www.catb.org/~esr/writings/cathedral-bazaar/) - Eric S. Raymond's landmark essay on OSS development models.
- [Open Source Guides](https://opensource.guide/) - GitHub's official guides for open source contributors.
- [Choose a License](https://choosealicense.com/) - Simplified license picker for open source projects.

### 🏛️ Governance & Community
- [Contributor Covenant](https://www.contributor-covenant.org/) - The most widely adopted code of conduct for open source projects.
- [OSI (Open Source Initiative)](https://opensource.org/) - The steward of the Open Source Definition.

### 🎓 Educational Resources
- [GitHub Learning Lab](https://lab.github.com/) - Hands-on learning for GitHub and open source.
- [freeCodeCamp](https://www.freecodecamp.org/) - Free coding curriculum used worldwide.

### ⚖️ Legal & Licensing
- [SPDX License List](https://spdx.org/licenses/) - Standardized list of open source licenses.
- [TLDRLegal](https://tldrlegal.com/) - Plain-English summaries of software licenses.
